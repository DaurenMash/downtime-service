Информация о многопоточности

Система использует ConcurrentHashMap для потокобезопасного хранения статусов оборудования и ScheduledExecutorService для запуска мониторинга в отдельных потоках.

  • Каждое оборудование мониторится в отдельном потоке
  • Статусы меняются случайным образом каждые 5-15 секунд
  • Используется потокобезопасная карта для доступа к данным
  • Автоматическое логирование статистики каждые 30 секунд
